# 상담 챗봇 데모 시나리오 (9개 패턴, 청년 특화 버전)

공통 규칙
- 질문 순서: situation → emotion → behavior → impact → duration → coping → goal → cause → thought
- thought/emotion/behavior에서 "모르겠다" → 재질문 1회 → 그래도 모르겠다면 "잘 모르겠음"으로 채움
- 나머지 6개 슬롯에서 "모르겠다" → 바로 "-"으로 채움
- off-topic 3연속 시 "그만 상담으로 돌아갈까요?" 유도

---

## 1. mood_energy_fluctuation_loop — off-topic 삽입
> 기분과 에너지가 크게 오르내리는 패턴

- 상담사(situation): 오늘 어떤 마음이나 상황 때문에 이야기해보고 싶으셨나요?
- 나: 창업 동아리에서 아이디어 하나가 통할 것 같으면 며칠은 뭐든 다 할 수 있을 것 같다가, 갑자기 하루는 완전히 가라앉아요.
- 상담사(emotion): 그때 드는 감정은 어떤가요?
- 나: 근데 어제 축구 보셨어요? 손흥민 골 미쳤던데. *(off-topic, 1턴)*
- 상담사: (잡담 응답, 질문 없음) 손흥민 선수 골이 화제였죠.
- 나: 아, 잠깐 다른 이야기를 했네요. 다시 상담하고 싶어요.
- 상담사(emotion, 재시도): 아까 말씀하신 기분 변화에서, 드는 감정은 어떤가요?
- 나: 들뜨고 자신만만하다가 갑자기 축 처지고 그래요.
- 상담사(behavior): 그럴 때 행동은 어떤가요?
- 나: 잠을 거의 안 자고 사업계획서를 밤새 쓰다가, 갑자기 팀원들한테 투자 얘기까지 꺼내버려요.
- 상담사(impact): 이게 일상에 어떤 영향을 주나요?
- 나: 동아리 사람들이 저보고 갑자기 딴사람 같다고 해요, 팀 분위기도 좀 이상해졌어요.
- 상담사(duration): 언제부터, 얼마나 자주 그런가요?
- 나: 한 달쯤 됐고, 며칠 간격으로 반복돼요.
- 상담사(coping): 그동안 해보신 방법이 있나요?
- 나: 딱히 없어요.
- 상담사(goal): 어떤 모습이 되면 좋겠어요?
- 나: 기분이 좀 안정적이었으면 좋겠어요.
- 상담사(cause): 왜 이렇게 됐다고 생각하세요?
- 나: 잘 모르겠어요, 그냥 그렇게 됐어요.
- 상담사(thought): 그럴 때 스스로 드는 생각은요?
- 나: 이 아이템만 밀어붙이면 인생이 완전히 달라질 것 같은 근거 없는 확신이 들어요.

**매칭 패턴**: mood_energy_fluctuation_loop / self-help: 충동적 결정 자제 관련 항목 등 (coping="없음"이라 제외 항목 없음)

## 2. executive_dysfunction_procrastination_loop — "모르겠다"(cause) 삽입
> 시작 어려움과 미루기 패턴

- situation: 자소서를 계속 미루다가 지원 마감 직전에야 겨우 손을 대게 돼요.
- emotion: 스스로한테 화도 나고 답답해요.
- behavior: 자소서 항목도 헷갈리고 마감 일정도 자주 깜빡해요.
- impact: 이번에도 지원한 곳 중 절반은 마감을 놓쳐버렸어요.
- duration: 두 달 정도 됐고 거의 매번 그래요.
- coping: 캘린더 알람도 맞춰보고 스터디 그룹도 들어가 봤는데 소용없었어요.
- goal: 미리미리 좀 준비하는 사람이 되고 싶어요.
- 상담사(cause): 왜 이렇게 됐다고 생각하세요?
- 나: 음... 잘 모르겠어요.
- (재질문 없음, "잘 모르겠음"으로 바로 채움 — cause는 재질문 대상 아님)
- thought: 이번에도 결국 서류를 못 낼 것 같다는 생각이 계속 들어요.

**매칭 패턴**: executive_dysfunction_procrastination_loop

## 3. low_mood_inertia_loop — 기본 흐름 (삽입 없음)
> 무기력과 활동 감소의 반복

- situation: 취업 준비하다가 계속 떨어지고 나니 무기력해서 아무것도 하기 싫고 계속 누워만 있어요.
- emotion: 우울하고 속이 텅 빈 것 같아요.
- behavior: 스터디 모임도 다 나가고 취미도 손 놨어요.
- impact: 이력서 쓰는 것도 다 미루게 됐어요.
- duration: 한두 달 정도, 거의 매일이에요.
- coping: 운동해보려 했는데 얼마 못 갔어요. 효과도 없었고요.
- goal: 그냥 다시 자소서든 뭐든 시작할 힘이 생겼으면 좋겠어요.
- cause: 계속 떨어지다 보니 그냥 지쳐서 그런 것 같아요.
- thought: 뭘 해도 결국 안 될 것 같다는 생각이 들어요.

**매칭 패턴**: low_mood_inertia_loop / self-help: coping에서 "운동"류 제외, goal("다시 시작") 부합 항목 우선

## 4. sleep_appetite_rhythm_disruption — "모르겠다"(thought, 재질문) 삽입
> 수면, 식사, 몸의 리듬이 흔들리는 패턴

- situation: 편의점 야간 알바를 시작하고 나서 밤낮이 바뀌고 끼니도 자주 걸러요.
- emotion: 예민하고 기운이 하나도 없어요.
- behavior: 낮에는 잠을 못 자거나 하루종일 자고, 머리도 자주 아파요.
- impact: 낮에 듣는 전공 수업 시간에 계속 졸아요.
- duration: 3주 정도 됐고 거의 매일이에요.
- coping: 알바 끝나고 일찍 자보려 했는데 잘 안 됐어요.
- goal: 생활 리듬을 다시 찾고 싶어요.
- cause: 야간 근무 스케줄 때문인 것 같아요.
- 상담사(thought): 그럴 때 스스로 드는 생각은요?
- 나: 잘 모르겠어요.
- 상담사(재질문): 어렵게 생각하지 마시고, 드는 생각을 편안하게 말씀해주시면 됩니다. 지금 겪고 계신 상황에서 스스로 하는 생각들을 이야기해주세요.
- 나: 음... 몸이 계속 무겁고 리듬을 도저히 못 찾겠다는 생각 정도요.

**매칭 패턴**: sleep_appetite_rhythm_disruption

## 5. overload_burnout_loop — off-topic(1턴, extraction 오버라이드) 삽입
> 부담 누적과 소진의 반복

- situation: 인턴 업무랑 졸업논문을 같이 하다 보니 할 일이 산더미인데 더는 못 버틸 것 같아요.
- emotion: 짓눌리는 느낌이고 별것도 아닌 일에 짜증이 나요.
- 나: 참, 저 다음 주에 토익 시험도 있어요. 그것도 준비해야 하는데 시간이 없어요.
- impact: 일상 리듬이 무너지고 팀원들과 부딪히는 일도 늘었어요. 또, 무리해서 계속 일하다가 결국 인턴 업무도 논문도 다 손 놓아버려요.
- duration: 한 달 전부터, 거의 매일이에요.
- coping: 하루 쉬어보려 했는데 죄책감만 들었어요.
- goal: 숨 쉴 틈이라도 있었으면 좋겠어요.
- cause: 인턴이랑 논문이 같은 시기에 몰려서 그런 것 같아요.
- thought: 이걸 다 해내지 않으면 큰일 날 것 같은 부담이 계속 들어요.

**매칭 패턴**: overload_burnout_loop

## 6. meaning_loss_acceptance_loop — "모르겠다"(goal) 삽입
> 상실 이후 의미를 찾기 어려운 패턴

- situation: 오랫동안 준비하던 공무원 시험이 최종에서 떨어지고 나서 삶의 의미를 잘 모르겠어요.
- emotion: 슬프고 마음이 텅 빈 것 같아요.
- behavior: 혼자 방에 틀어박혀서 같은 생각만 반복해요.
- impact: 스터디원들과의 연락도 점점 뜸해졌어요.
- duration: 석 달 정도 됐고 계속 그래요.
- coping: 다른 진로 상담을 좀 받아봤는데 크게 달라지진 않았어요.
- 상담사(goal): 어떤 모습이 되면 좋겠어요?
- 나: 잘 모르겠어요.
- (6개 슬롯 중 하나이므로 재질문 없이 바로 "잘 모르겠음"으로 채움)
- cause: 왜 이렇게 됐는지는 저도 잘 설명이 안 돼요.
- thought: 이 몇 년을 다 쏟아부었는데 받아들일 수가 없다는 생각뿐이에요.

**매칭 패턴**: meaning_loss_acceptance_loop

## 7. negative_self_evaluation_loop — off-topic 3연속(복귀 거부 후 재복귀) 삽입
> 자책과 부정적 생각의 반복

- situation: 조별과제에서 실수할 때마다 다 제 탓이라는 생각이 심해져요.
- emotion: 죄책감이랑 부끄러움이 계속 들어요.
- 나: 근데 저희 학교 축제 얘기 들으셨어요? *(off-topic 1)*
- 상담사: (리액션만) 축제 얘기가 있었군요.
- 나: 이번에 유명한 가수도 온대요. *(off-topic 2)*
- 상담사: (리액션만) 기대되시겠어요.
- 나: 티켓 구하기 힘들 것 같아요. *(off-topic 3 → 복귀 유도)*
- 상담사: 티켓 구하기 힘드시겠어요. 그런데 지금 대화는 상담을 위한 것이 아니라 다른 주제였는데, 혹시 괜찮다면 상담으로 복귀해도 될까요?
- 나: 아니요, 조금만 더 얘기하고 싶어요. *(복귀 거부 → 카운터 0 리셋)*
- 나: 축제 때 부스도 있대요. *(off-topic 1, 리셋 후 재시작)*
- 상담사: (리액션만) 부스도 다양하게 있나 보네요.
- 나: 아 근데 이제 상담 다시 할게요. *(resume_intent=True, 카운터와 무관하게 즉시 복귀)*
- 상담사(behavior, 재개): 자책하는 마음이 들 때 어떤 행동을 하게 되나요?
- 나: 조별과제에서 의견 내는 것도 그만두고 발표 같은 것도 다 피해요.
- impact: 새로운 팀플에 들어가는 게 다 두려워졌어요.
- duration: 두 달 정도, 거의 매번 그래요.
- coping: 스스로를 다독여보려 했는데 오래 안 갔어요.
- goal: 제 자신을 좀 덜 몰아붙이고 싶어요.
- cause: 완벽해야 한다는 생각 때문인 것 같아요.
- thought: 저는 뭘 해도 결국 팀에 민폐만 끼치는 사람 같다는 생각이 들어요.

**매칭 패턴**: negative_self_evaluation_loop

## 8. intrusive_thought_loop — "모르겠다"(emotion, 재질문) 삽입
> 원치 않는 생각이 반복해서 떠오르는 패턴

- situation: 단톡방에 메시지를 보내고 나면 실수한 표현이 있었는지 자꾸 불쑥 떠올라서 힘들어요.
- 상담사(emotion): 그럴 때 드는 감정은요?
- 나: 잘 모르겠어요, 그냥 이상해요.
- 상담사(재질문): 어렵게 생각하지 마시고, 느끼는 감정을 편안하게 말씀해주시면 됩니다. 지금 겪고 계신 상황에서 드는 감정들을 이야기해주세요. 압박감, 공허함, 자신감, 슬픔 어떤 감정이든지요.
- 나: 아, 찜찜하고 불안한 것 같아요.
- behavior: 보낸 메시지를 몇 번이고 다시 읽어보게 돼요.
- impact: 과제나 다른 일도 자꾸 미루게 돼요.
- duration: 두 달 정도, 하루에도 여러 번 그래요.
- coping: 생각을 안 하려고 다른 데 집중해봤는데 잘 안 됐어요.
- goal: 그 생각에 덜 휘둘리고 싶어요.
- cause: 왜 이런지는 저도 잘 모르겠어요.
- thought: 원하지도 않는 생각이 자꾸 머리에 떠올라요.

**매칭 패턴**: intrusive_thought_loop

## 9. trauma_intrusion_avoidance_loop — off-topic(오탐 후 자체 교정) 삽입
> 충격 기억과 회피의 반복

- situation: 작년에 자취방 근처에서 교통사고를 겪은 게 계속 떠올라서 힘들어요.
- emotion: 무섭고 몸이 긴장돼요.
- 나: 그때 이후로 그 사거리는 아예 안 지나가요. *(off-topic으로 오분류될 뻔했으나 extraction이 behavior 정보를 찾아내 자동 교정 — 질문 건너뛰지 않고 정상 진행)*
- 상담사(impact): 이게 일상에 어떤 영향을 주나요?
- 나: 통학할 때 그 근처로는 아예 못 가고 버스도 일부러 돌아서 타요.
- duration: 반년 전부터, 비슷한 상황이 생기면 항상 그래요.
- coping: 학교 상담센터를 가본 적이 있는데 중간에 그만뒀어요.
- goal: 그 기억에 덜 휘둘리고 싶어요.
- cause: 그때 그 사고 때문인 게 분명해요.
- thought: 또 그런 일이 생기지 않을까 하는 두려움이 계속돼요.

**매칭 패턴**: trauma_intrusion_avoidance_loop / self-help: coping에서 "상담 중단" 언급 있으므로 professional_consultation_and_monitoring 제안 문구 조정 필요할 수 있음

## 10. 슬롯 충분성(sufficient) 체크 — situation, 재질문 후 해결
> "채워지긴 했지만 내용이 부실한" 답변에 대한 thin_retry 동작 확인용

- 상담사(situation): 오늘 어떤 마음이나 상황 때문에 이야기해보고 싶으셨나요?
- 나: 그냥 요즘 취업 준비 때문에 힘들어서요.
- *(slot_value 채워짐, 하지만 sufficient=false — missing_aspect: 너무_포괄적. thin_retry_count[situation]=0이므로 재질문 진행, 다음 슬롯으로 넘어가지 않음)*
- 상담사(thin_retry): 취업 준비하시면서 구체적으로 어떤 일이 있으셨는지 조금 더 말씀해주실 수 있을까요?
- 나: 아, 취업 스터디에서 저만 계속 서류에서 떨어지고 있어요. 스터디원들은 다 면접까지 가는데 저만 못 가서요.
- *(재추출 결과 slot_value 갱신, sufficient=true — 다음 슬롯으로 정상 진행)*
- 상담사(emotion): 그때 드는 감정은 어떤가요?
- 나: 창피하고 제가 부족한 사람처럼 느껴져요.
- behavior: 스터디 모임에 나가는 것도 점점 꺼려지고 피하게 돼요.
- impact: 스터디원들 얼굴 보기도 민망해서 모임 자체를 그만둘까 고민 중이에요.
- duration: 두 달 정도 됐고, 서류 결과 나올 때마다 반복돼요.
- coping: 자소서 첨삭을 더 받아봤는데 크게 달라지진 않았어요.
- goal: 스터디원들 눈치 안 보고 편하게 준비하고 싶어요.
- cause: 제가 남들보다 준비가 부족해서 그런 것 같아요.
- thought: 나만 유독 뒤처지는 사람 같다는 생각이 반복돼요.

**매칭 패턴**: negative_self_evaluation_loop

## 11. 슬롯 충분성(sufficient) 체크 — emotion, 재질문에도 여전히 부실 → 그대로 채택
> 재질문 1회 소진 후에도 thin이면 있는 값 그대로 채택하고 다음 슬롯 진행하는 동작 확인용

- situation: 편의점 알바하다가 진상 손님한테 크게 혼난 뒤로 계속 신경 쓰여요.
- 상담사(emotion): 그때 드는 감정은 어떤가요?
- 나: 그냥 힘들어요.
- *(slot_value="힘들어요", sufficient=false — missing_aspect: 포괄적_감정어. thin_retry_count[emotion]=0이므로 재질문)*
- 상담사(thin_retry): 그 순간에 조금 더 구체적으로 어떤 감정이셨는지 말씀해주실 수 있을까요? 예를 들어 속상함, 억울함, 부끄러움 같은 것들이요.
- 나: 음... 그냥 다 그런 것 같아요.
- *(재추출해도 여전히 sufficient=false. 그러나 thin_retry_count[emotion]이 이미 1이므로 더 캐묻지 않고 "그냥 다 그런 것 같아요"를 emotion 값으로 그대로 채택, 다음 슬롯 진행)*
- 상담사(behavior): 그럴 때 어떤 행동을 하게 되나요?
- 나: 그 손님이 다시 올까 봐 계산대 서는 것 자체가 꺼려져요.
- impact: 알바 스케줄 짤 때도 자꾸 그 시간대를 피하게 돼요.
- duration: 3주 정도 됐고, 비슷한 손님만 봐도 그래요.
- coping: 딱히 해본 건 없어요.
- goal: 손님 응대할 때 좀 덜 위축됐으면 좋겠어요.
- cause: 그때 제대로 대처를 못 해서 그런 것 같아요.
- thought: 또 그런 손님이 오면 어쩌나 하는 생각이 계속 들어요.

**매칭 패턴**: trauma_intrusion_avoidance_loop (경미한 형태) — emotion 슬롯이 재질문 후에도 thin인 채로 남았지만, 시스템은 정상적으로 다음 슬롯으로 진행하고 최종 매칭도 나머지 정보로 이루어짐을 보여주는 사례

## 12. 슬롯 충분성(sufficient) 체크 — behavior, 재질문 후 해결
> missing_aspect가 "행동_없이_상태만"인 경우의 thin_retry 확인용

- situation: 조별과제에서 발표를 맡았는데 그날 이후로 비슷한 자리가 계속 부담스러워요.
- emotion: 불안하고 심장이 두근거려요.
- 상담사(behavior): 그럴 때 어떤 행동을 하게 되나요?
- 나: 그냥 버텨요.
- *(slot_value="그냥 버텨요", sufficient=false — missing_aspect: 행동_없이_상태만. thin_retry_count[behavior]=0이므로 재질문)*
- 상담사(thin_retry): 그 상황에서 실제로 어떻게 하시는지 조금 더 구체적으로 말씀해주실 수 있을까요?
- 나: 아, 발표 순서가 다가오면 화장실 핑계 대고 자리를 피하거나, 아예 대본을 다 외워서 눈도 안 마주치고 읽어버려요.
- *(재추출 결과 sufficient=true — 다음 슬롯으로 정상 진행)*
- impact: 조별과제에서 발표 역할을 아예 안 맡으려고 피해 다녀요.
- duration: 한 학기 정도, 발표 있을 때마다 그래요.
- coping: 미리 대본을 여러 번 연습해봤는데 그날 되면 소용없었어요.
- goal: 발표할 때 덜 긴장하고 싶어요.
- cause: 예전에 발표하다가 크게 실수한 적이 있어서 그런 것 같아요.
- thought: 또 실수해서 사람들 앞에서 망신당할 것 같은 생각이 계속 들어요.

**매칭 패턴**: negative_self_evaluation_loop / trauma_intrusion_avoidance_loop 경계 사례 — behavior가 처음엔 "그냥 버텨요"로 뭉뚱그려졌으나 재질문으로 구체적 회피 행동(자리 피함, 눈 안 마주침)이 드러나 매칭 정확도가 올라가는 걸 보여주는 사례
